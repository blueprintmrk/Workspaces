How to use Chrome

Edit or delete bookmarks

Drag and drop bookmarks to move them around on the bookmarks bar. To make changes to a bookmark’s name, URL, or folder, edit it directly on the bookmarks bar. Or use the bookmark manager if you need to organize multiple bookmarks at once. 

Accessing the bookmark bar or bookmark manager

  • The bookmark bar is docked underneath the address bar by default. If you don’t see it, you can always find it by opening a new tab.
  • To open the bookmarks manager, click the wrench icon on the browser toolbar and go to Bookmarks > Bookmark manager.

Move bookmarks to another location

  • Move a single bookmark: Simply drag it to its new destination on the bookmarks bar. Keep in mind that bookmarks can only be in one folder at a time.
  • Move multiple bookmarks: Use the bookmarks manager and hold the Shift key while selecting the items you want to move. You can then move them all at the same time to their new destination.

Edit bookmark details

  1. Find your bookmarks on the bookmarks bar or bookmark manager.
  2. Right-click the bookmark that you’d like to edit.
  3. Select Edit from the menu.
  4. You can update the bookmark’s name, URL, and folder in the Edit Bookmark dialog that appears.

Delete bookmarks

  1. Find your bookmarks on the bookmarks bar or bookmark manager.
  2. Right-click the bookmark that you’d like to delete.
  3. Select Delete from the menu. If you delete a bookmark folder, you also delete all the bookmarks contained within the folder.

If you don’t want to see a bookmark on the bookmarks bar, you can store it in the "Other bookmarks" folder. This folder can't be deleted, but won’t show if it's empty. When it’s hidden, you can still add bookmarks to the folder using the bookmark manager.

If your bookmarks are synced to your Google Account, all the changes you make to your bookmarks will instantly show up on other devices where you’re signed in to Chrome. Learn more about Signing in to Chrome
