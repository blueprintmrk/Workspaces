Validation Status: Action Required

The validation process cannot continue because your web browser may not be configured to run the required script. In order to continue with validating your software, please try one of the two options below:

Option 1

Adjust Browser Settings to Enable Active Scripting

  1. Open Internet Explorer
  2. Click on the Tools menu and select Internet Options
  3. On the Security tab, click the Custom Level button.
  4. Scroll down the Security Settings list until you see Scripting and Active scripting.
  5. Select Enable and click OK, this will close the Security Settings dialog box.
  6. Click OK on the open Internet Options dialog box to close and save the changes just made.


Are you ready to try again?

After you have completed the steps above and your security settings have been modified, continue on to run the test again to make sure that the problem has been resolved.

Validation Status: Action Required

The validation process cannot continue because your web browser may not be configured to accept cookies.

Adjust Browser Settings to Enable Cookies

  1. Open Internet Explorer
  2. Click on the Tools menu and select Internet Options
  3. On the Privacy tab, move the settings slider to Low or Accept All Cookies
  4. Click OK

Are you ready to try again?

After you have completed the steps above and your privacy settings have been modified, continue on to run the test again to make sure that the problem has been resolved.
